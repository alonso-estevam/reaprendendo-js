Introdução

JavaScript é uma linguagem de tipagem dinâmica - você não precisa explicitar o tipo de dado ao declarar e inicializar uma variável, e também pode reatribuir o valor para outro tipo de dado sem causar erro. Além disso, é fracamente tipada, ou seja, "permite a conversão implícita de tipo quando uma operação envolve tipos incompatíveis".

JS interno e externo

Para fazer o JavaScript interagir com o html, podemos colocar o código dentro de scripts internos ou externos.

O interno vai dentro do próprio arquivo html, dentro de um trecho de código limitado pela tag <script> e </script>

Porém, o mais comum é criar um arquivo externo com a extensão .js e depois referenciar o arquivo dentro do html com a tag <script src="arquivoJavaScript.js"></script>. As duas formas funcionam da mesma maneira.
